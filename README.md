## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/nesh1205/thinesh-portfolio.git
```

### Navigate to Project

```bash
cd thinesh-portfolio
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build Production Version

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
